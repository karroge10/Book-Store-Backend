# Book Store Backend
 Backend build with Node.js and MySQL

## Использование

### `npm start`
Для запуска.
Открыть [http://localhost:3000](http://localhost:3000) чтобы посмотреть в браузере.

### `npm run dev`
Для запуска в режиме разработки.
Открыть [http://localhost:3000](http://localhost:3000) чтобы посмотреть в браузере.
