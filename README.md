# Book Store Backend
 Backend build with Node.js and MySQL

## Использование

### Подключится к Apache и MySQL в XAMPP.
Для запуска сервера и базы данных.

### `npm start`
Для запуска.
[http://localhost:3000](http://localhost:3000) в браузере.

### `npm run dev`
Для запуска в режиме разработки.
[http://localhost:3000](http://localhost:3000) в браузере.
